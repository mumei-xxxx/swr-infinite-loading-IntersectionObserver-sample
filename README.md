## 環境の立ち上げ

1. Node.js のバージョンを設定。（`.node-version` に記載）

2. pnpm をインストール

  - https://pnpm.io/ja/installation

3. 以下でインストール。

```
pnpm install
```

4. 以下で開発モードでサーバーが立ち上がる。

```
pnpm run dev
```
